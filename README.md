Loan Prediction Machine Learning Project

This project predicts loan approval status using real loan-related features. It demonstrates end-to-end machine learning workflow including data cleaning, feature selection, model training, and evaluation.

Features

The dataset includes:

person_age – Age of the applicant

person_income – Monthly income of the applicant

person_emp_exp – Employment experience (years)

loan_amnt – Loan amount requested

credit_score – Applicant’s credit score

loan_status – Target variable (approved/rejected)

Project Overview

Data Cleaning – Handling missing values and incorrect entries.

Feature Selection – Selecting key features that affect loan approval.

Model Training – Training machine learning models to predict loan status
